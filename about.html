About this project

This project was created for IT 3203: Intro to Web Development at Kennesaw State University. The topic was selected because HTTP and HTTPS connect web development with cybersecurity, browser behavior, networking, and user trust.

The site was redesigned for Milestone 3 to improve responsive layout, mobile navigation, research depth, quiz functionality, code organization, and documentation.

Author

Ryan Mangunda
Kennesaw State University
IT major with a cybersecurity focus

The project uses original writing, standards-based research, responsive HTML/CSS, and JavaScript for navigation and quiz interaction.

Design choices

  • Semantic HTML elements such as header, nav, main, section, article, figure, and footer.
  • A single external CSS file for consistent layout, colors, spacing, typography, cards, tables, forms, and responsive behavior.
  • A single external JavaScript file for the mobile menu and quiz logic.
  • Responsive tables that can scroll horizontally on small screens instead of breaking the page layout.
  • Accessible images with descriptive alt text and visible focus styles for keyboard navigation.
