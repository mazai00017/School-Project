Research resources with comments

The resources below support the research pages and key concepts. Standards documents are included because HTTP, HTTPS, QUIC, and TLS are technical protocols, so primary specifications matter.

MDN Web Docs: Overview of HTTP

Used for the basic client-server explanation, request-response flow, and HTTP position in the web stack.

MDN Web Docs: HTTP

Used for HTTP definitions, statelessness, methods, headers, and general developer reference material.

MDN Web Docs: Evolution of HTTP

Used for the historical movement from early HTTP to HTTP/1.1, HTTP/2, and HTTP/3.

IETF RFC 9110: HTTP Semantics

Used for the standards-based explanation of HTTP semantics shared across versions.

IETF RFC 9111: HTTP Caching

Used for explaining why caching is part of HTTP performance and reliability.

IETF RFC 9112: HTTP/1.1

Used for technical background on HTTP/1.1 message syntax and connection behavior.

IETF RFC 9113: HTTP/2

Used for HTTP/2 features such as binary framing, multiplexing, and header compression.

IETF RFC 9114: HTTP/3

Used for HTTP/3 structure, QUIC mapping, streams, and protocol goals.

IETF RFC 9000: QUIC

Used for QUIC background, UDP transport behavior, stream multiplexing, and TLS integration.

IETF RFC 8446: TLS 1.3

Used for explaining modern TLS security concepts behind HTTPS and QUIC.

web.dev: Why HTTPS Matters

Used for explaining why HTTPS protects integrity, privacy, and modern browser feature access.

web.dev: Enable HTTPS on Your Servers

Used for certificate, CSR, key, and deployment best-practice explanation.

Let’s Encrypt: How It Works

Used for practical explanation of certificate issuance and domain validation.

HTTP Archive Web Almanac 2025

Used as a current web-industry source for understanding how the modern web is measured and analyzed.
